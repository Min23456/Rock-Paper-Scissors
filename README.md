# ✊✋✌️ Rock Paper Scissors

Rock Paper Scissors is a classic, fast-paced hand game turned into a simple digital experience. Play against the computer, test your luck, and sharpen your decision-making skills in this timeless game of strategy and chance.

---

## 🌐 Live Demo

👉 **Play Here:**
[PLAY NOW!](https://min23456.github.io/Rock-Paper-Scissors/)

## 🎮 How to Play

* Choose **Rock**, **Paper**, or **Scissors**.
* The computer will randomly choose its move.
* The winner is decided instantly:

  * Rock beats Scissors
  * Scissors beats Paper
  * Paper beats Rock
* If both choices are the same, it’s a tie.
* Keep playing to improve your streak!

## 🧩 Game Logic Overview

* The player selects one of three options.
* The computer generates a random choice.
* The game compares both selections:

  * If player wins → score increases
  * If computer wins → computer score increases
  * If tie → no score change
* The result is displayed after each round.
